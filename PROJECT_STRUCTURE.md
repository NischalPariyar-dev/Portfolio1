# Project Structure Guide

This document outlines the recommended folder and file organization for the Portfolio1 website project.

## Directory Structure

```
Portfolio1/
│
├── index.html          # Main HTML file
├── about.html          # About page HTML file
├── projects.html       # Projects page HTML file
├── contact.html        # Contact page HTML file
│
├── css/               # Folder for CSS stylesheets
│   ├── styles.css      # Main stylesheet
│   ├── about.css       # Styles for about page
│   └── contact.css     # Styles for contact page
│
├── js/                # Folder for JavaScript files
│   ├── scripts.js      # Main JavaScript file
│   └── utils.js       # Utility functions
│
├── images/            # Folder for images
│   ├── logo.png       # Project logo
│   └── project1.png   # Example project image
│
├── docs/              # Folder for documentation
│   └── README.md      # Project overview and setup instructions
│
└── assets/            # Folder for additional assets (fonts, icons, etc.)
    ├── fonts/
    └── icons/
```

## Explanation of Folders
- **HTML files**: Contain the main content of the web pages.
- **css/**: Contains stylesheets that control the appearance of the website.
- **js/**: Contains JavaScript files for interactive functionality.
- **images/**: Holds all images used in the project.
- **docs/**: Contains project-related documentation such as README and project guidelines.
- **assets/**: For additional resources like fonts and icons used in the project.

This structure helps maintain organization, making it easy to find and manage different aspects of the project.
